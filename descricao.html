Descrição das Bases Numéricas

Base Decimal

A base decimal é a que utilizamos no nosso dia a dia, composta por dez dígitos de 0 a 9. É amplamente usada em cálculos e medições.

Passo a Passo para Conversão de Decimal para Decimal:

  1. Não há conversão necessária, pois é a base padrão.
  2. Os números permanecem os mesmos.

Exemplo: O número "23" já está em decimal.

Base Binária

A base binária utiliza apenas dois dígitos: 0 e 1. É fundamental em computação, pois os computadores processam informações em formato binário.

Passo a Passo para Conversão de Decimal para Binário:

  1. Divida o número decimal por 2.
  2. Registre o quociente e o resto.
  3. Continue dividindo o quociente por 2 até que ele seja 0.
  4. Os restos, lidos de baixo para cima, formam o número binário.

Exemplo: Converter 5 para binário:

5 ÷ 2 = 2, resto 1
2 ÷ 2 = 1, resto 0
1 ÷ 2 = 0, resto 1
Portanto, 5 = 101 (binário)
Base Hexadecimal

A base hexadecimal utiliza dezesseis símbolos, de 0 a 9 e de A a F. É frequentemente utilizada em programação e design de cores.

Passo a Passo para Conversão de Decimal para Hexadecimal:

  1. Divida o número decimal por 16.
  2. Registre o quociente e o resto.
  3. Continue dividindo o quociente por 16 até que ele seja 0.
  4. Os restos, lidos de baixo para cima, formam o número hexadecimal.

Exemplo: Converter 255 para hexadecimal:

255 ÷ 16 = 15, resto 15 (F)
15 ÷ 16 = 0, resto 15 (F)
Portanto, 255 = FF (hexadecimal)
Base Octal

A base octal é uma representação numérica que utiliza oito dígitos, de 0 a 7. É menos comum, mas é utilizada em alguns sistemas computacionais.

Passo a Passo para Conversão de Decimal para Octal:

  1. Divida o número decimal por 8.
  2. Registre o quociente e o resto.
  3. Continue dividindo o quociente por 8 até que ele seja 0.
  4. Os restos, lidos de baixo para cima, formam o número octal.

Exemplo: Converter 10 para octal:

10 ÷ 8 = 1, resto 2
1 ÷ 8 = 0, resto 1
Portanto, 10 = 12 (octal)
